Icon

Load icons inside your Grommet application.

Properties

colorIndex {category}-{index}
The color identifier to use for the stroke color. For example: "brand". Use "plain" to keep the original icon colors.
size xsmall|small|medium|large|xlarge|huge
The icon size. Defaults to small.
type control|logo|status
The type of the icon. Defaults to control.

Usage

import CloudIcon from 'grommet/components/icons/base/Cloud';
//or var CloudIcon = Grommet.Icons.Base.Cloud;

<CloudIcon />
